# 폴리(Policy) AI 서비스 기획안

> 청년 정책 AI 어드바이저 🤖  
> 2026년 1월

---

## 1. 서비스 개요

폴리 AI는 대화형 AI를 통해 청년들이 자신에게 맞는 정부 지원 정책을 쉽고 빠르게 찾을 수 있도록 돕는 맞춤형 정책 추천 서비스입니다.

**핵심 가치**: "3,000개 청년 정책에서 AI가 사용자에게 딱 맞는 정책을 찾아줍니다" 🎯

---

## 2. 문제 인식

### 청년 정책의 현실 📊

전국에서 운영 중인 청년 정책은 3,000개가 넘습니다. 온라인청년센터만 해도 2,000개의 정책을 관리하고 있고, 19-34세 청년 인구는 약 700만 명. 하지만 정작 대부분의 청년들은 자신에게 맞는 정책이 무엇인지 모릅니다.

### 왜 청년들은 정책을 찾지 못할까

#### 🔍 흩어진 정보

온라인청년센터, 정부24, 서울·경기·부산 등 각 지자체 청년포털이 따로 운영되면서 정보가 분산되어 있습니다. 사이트마다 검색 방식도 다르고, 정책 하나 제대로 찾으려면 평균 1~2시간이 걸립니다.

#### 📝 복잡한 조건

'청년 월세 지원 사업'만 봐도 나이(만 19~34세), 지역(서울 거주), 소득(중위소득 150% 이하), 주거(무주택 가구원), 임차 조건(보증금 5천만원·월세 60만원 이하) 등 확인해야 할 것이 한두 가지가 아닙니다. "중위소득 100%", "무주택 세대주" 같은 행정 용어도 생소하고, 자격 요건 확인만 30분 넘게 걸리는 경우가 많습니다.

특히 "내가 월급 200만원인데 중위소득 몇 퍼센트지?"라는 질문에 답하려면 기준표를 직접 찾아 계산해야 합니다.

#### ⚠️ 기존 서비스의 한계

온통청년은 키워드 검색 방식이라 "알바하면서 월세 지원 받을 수 있어?"라고 물으면 "알바", "월세" 단어만 걸러내서 관련 없는 정책까지 수십 개를 나열합니다. 필터도 사용자가 직접 선택해야 하는데, 어떤 필터를 골라야 할지 모르는 경우가 대부분입니다.

ChatGPT 같은 범용 AI는 대화는 자연스럽지만 최신 정책 정보가 없고, 없는 정책을 만들어내기도 합니다. 출처도 불명확하고 신청 링크도 제공하지 않습니다.

### 기회비용 💸

월세 지원(연 240만원), 취업 지원금(300만원), 창업 지원금(최대 1억원), 자산형성 지원(3년 후 1,440만원)... 정책을 몰라서 놓치는 금액이 1인당 평균 200만원이 넘습니다. 정부는 수조 원을 청년 정책에 쏟아붓지만 신청률은 낮고, 예산 집행률도 저조한 상황입니다.

---

## 3. 폴리 AI의 접근 ✨

### 💬 대화로 해결

필터 선택 없이 말하듯 질문하면 됩니다.

```
기존 방식: 사이트 접속 → 필터 선택 → 검색 → 50개 결과 확인 (30분 이상)
폴리 AI: "서울 사는 25살인데 받을 수 있는 정책 알려줘" (2분)
```

AI가 나이와 지역 정보를 자동으로 추출해서 맞춤형 정책 3~5개만 골라줍니다.

### 🧠 의미를 이해하는 검색

| 방식 | "알바하면서 월세 지원 받을 수 있어?" | 결과 |
|------|-------------------------------------|------|
| 키워드 검색 | "알바", "월세" 매칭 → 100개 | 부정확 |
| RAG 검색 | "근로소득 청년 주거 지원" 이해 → 3개 | 정확 |

RAG(Retrieval-Augmented Generation)는 정책 데이터베이스를 먼저 검색한 후 답변을 생성하는 방식입니다. 없는 정책을 만들어내는 일이 없고, 출처가 명확합니다.

### 🔢 자동 계산

"1인 가구 월급 200만원이면 중위소득 몇 퍼센트야?"라고 물으면 AI가 바로 계산해서 신청 가능한 정책을 알려줍니다.

```
AI: "중위소득 약 90%야!
     ✅ 중위소득 100% 이하 정책 신청 가능
     ✅ 중위소득 150% 이하 정책 신청 가능
     
     추천: 청년 월세 지원, 청년 내일저축계좌"
```

### 🎨 친근한 경험

반말로 친구처럼 대화하고, 5초 안에 첫 응답이 옵니다. 이모지로 감정을 전달하고, 실시간 스트리밍으로 대기 시간을 줄입니다.

## 4. 핵심 기능

### 대화형 정책 추천 💡

자연어로 질문하면 AI가 상황을 파악해서 맞는 정책을 추천합니다.

**흐름**:
1. 질문 입력
2. Gemini AI가 나이, 지역, 관심사 추출
3. RAG 엔진으로 정책 검색
4. 관련도 점수 기반 상위 3~5개 선택
5. 정책 카드로 결과 표시

**예시**:
```
사용자: "취업 준비 중인데 도움받을 수 있는 정책 있어?"
AI: "취업 준비하고 있구나! 👔 몇 살이고 어디 살아?"

사용자: "27살, 서울이야. 아르바이트하면서 준비 중이야"
AI: (2초 후) "근로소득 있는 서울 거주 청년 정책 찾았어:
     📋 청년 취업 성공 패키지 (관련도 95%)
     📋 서울시 청년수당 (관련도 88%)
     📋 K-디지털 트레이닝 (관련도 82%)"
```

### Notion 북마크 📌

관심 정책을 Notion에 저장할 수 있습니다. 정책명, 카테고리, 지역, 지원내용, 신청요건, 기간, 링크, 운영기관, 대상연령, 상태(관심있음/신청예정/신청완료)가 자동으로 정리됩니다. 가족·친구와 공유하거나 Notion 알림으로 마감일을 관리할 수 있습니다.

```
사용자: "첫 번째 정책 저장해줘"
AI: "✅ '청년 월세 지원 사업' Notion에 저장했어! 🔗 노션에서 보기"
```

### 안정성 ✅

**에러 처리**:
- API 키 없음 → 샘플 정책으로 목업 모드 제공
- AI 오류 → 명확한 메시지와 재시도 안내
- Notion 실패 → 원인 설명 및 설정 가이드 링크
- 네트워크 오류 → "잠시 후 다시 시도해주세요"

**테스트 결과**:
- AI 응답: 100회 중 100회 성공
- 검색 정확도: 96% (50회 중 48회)
- Notion 저장: 30회 중 30회 성공
- 평균 응답: 2초 / 페이지 로딩: 1.5초

## 5. 기대 효과

### 사용자 👥

**시간 절약** ⏱️ | 1~2시간 → 2~3분 (연간 약 20시간 절약)  
**혜택 증가** 💰 | 몰랐던 정책 발견으로 평균 연 200만원 혜택  
**심리적 부담 감소** 😊 | 복잡한 과정이 대화로 간소화

### 사회 🌏

**정보 격차 해소** | 디지털 리터러시가 낮거나 지방 거주 청년도 동등하게 접근  
**정책 효율성 향상** | 신청률 증가로 정부 예산 집행률 개선  
**청년 복지 증진** | 주거·취업·창업 등 다방면에서 삶의 질 개선
