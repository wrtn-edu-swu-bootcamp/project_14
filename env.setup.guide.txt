===================================
폴리 AI 환경 변수 설정 가이드
===================================

프로젝트를 실행하려면 .env.local 파일을 생성하고 API 키를 설정해야 합니다.

1. .env.local 파일 생성
   프로젝트 루트 디렉토리에 '.env.local' 파일을 만드세요.

2. 아래 내용을 복사하여 .env.local 파일에 붙여넣으세요:

---복사 시작---
# OpenAI API Key (필수)
OPENAI_API_KEY=your_openai_api_key_here

# 온라인청년센터 API Key (선택사항)
YOUTH_CENTER_API_KEY=your_youth_center_api_key_here
---복사 끝---

3. API 키 발급 방법

[OpenAI API Key - 필수]
  1. https://platform.openai.com/ 접속
  2. 'API Keys' 메뉴에서 '+ Create new secret key' 클릭
  3. 생성된 키(sk-proj-로 시작)를 복사
  4. .env.local 파일의 your_openai_api_key_here 부분에 붙여넣기
  
  예시:
  OPENAI_API_KEY=sk-proj-abcdefghijk123456789

[온라인청년센터 API Key - 선택사항]
  1. https://www.data.go.kr/data/15143273/openapi.do 접속
  2. '활용신청' 버튼 클릭 (로그인 필요)
  3. 신청 후 승인 대기 (보통 1-2일 소요)
  4. 승인된 인증키를 .env.local에 추가
  
  참고: 이 키가 없어도 프로젝트는 목업 데이터로 작동합니다!

4. 서버 재시작
   .env.local 파일을 수정한 후에는 서버를 재시작해야 합니다.
   
   PowerShell에서:
   - Ctrl+C로 서버 중지
   - npm run dev로 다시 시작

===================================
문제 해결
===================================

Q: "OPENAI_API_KEY is not defined" 오류가 나요
A: .env.local 파일이 프로젝트 루트에 있는지 확인하고, 
   파일명이 정확히 '.env.local'인지 확인하세요.

Q: 목업 데이터가 계속 보여요
A: 온라인청년센터 API 키가 없거나 잘못된 경우 자동으로 
   목업 데이터를 사용합니다. 실제 데이터를 사용하려면 
   공공데이터포털에서 API 키를 발급받으세요.

Q: GPT-4 비용이 걱정돼요
A: app/api/chat/route.ts에서 모델을 'gpt-3.5-turbo'로 
   변경하면 비용을 절감할 수 있습니다.
